# bs-webapi

Experimental bindings to the DOM and other Web APIs.

## Installation
```
npm install bs-webapi
```
Then add `bs-webapi` to `bs-dependencies` in your `bsconfig.json`. A minimal example:
```
{
  "name": "my-thing",
  "sources": "src",
  "bs-dependencies": ["bs-webapi"]
}
```
